## BlE Sense

### Download Board

1. Go to Tools--> Board--> Manage Board--> Seach: mbed(Arduino Mbed OS Nano Board by Arduino)
2. Select Board Tools--> Board--> Arduino Mbed OS ...--> Arduino nano 33 BLE


<br>
<br>

### Installing Arduino Gesture Sensor library

1. Add library, Go to tools--> Manage libraries-->Search:APD (Arduino_APDS9960 by Arduino)
2. Select Sketch: Go to--> file--> Example-->Arduino_APDS9960-->Gesture Sensor
